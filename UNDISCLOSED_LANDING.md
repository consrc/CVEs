## Disclosure Policy

### Supported Products (Vendor Still Providing Updates)

For products that are actively maintained by the vendor:

- The vendor is provided reasonable time to investigate and develop a remediation.
- Public disclosure is deferred until:
  - a security patch or mitigation becomes available, or
  - the coordinated disclosure period expires.
- Once disclosure is authorized, the CVE reference URL will be modified to point to a page with:
  - the assigned CVE identifier,
  - affected version information,
  - vulnerability metadata,
  - and any associated proof-of-concept material.

---

### End-of-Service / Unsupported Products

For products confirmed to be end-of-service or no longer supported by the vendor:

- The vendor is still notified and provided an opportunity to respond.
- A response window of **15 days** is granted before public disclosure proceeds.
- Since no security updates are expected, disclosure may occur without a vendor patch and the CVE reference URL will be modified to point to a page with:
  - the assigned CVE identifier,
  - affected version information,
  - vulnerability metadata,
  - and any associated proof-of-concept material.
